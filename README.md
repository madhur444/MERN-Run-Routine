# 🚀 Habit Tracker

A modern Full Stack Habit Tracker application that helps users build consistency by creating, tracking, and managing daily habits.

---

## 📌 Features

- 🔐 User Authentication
- ➕ Create Habits
- ✏️ Update Habits
- 🗑️ Delete Habits
- ✅ Mark Habits as Completed
- 📅 Daily Habit Tracking
- 📊 Progress Dashboard
- 📱 Responsive UI
- 🌙 Modern Interface
- ⚡ Fast Performance

---

# 🛠️ Tech Stack

## Frontend
- HTML5
- CSS3
- JavaScript
- Vite

## Backend
- Node.js
- Express.js

## Database
- MongoDB

## Authentication
- JWT (JSON Web Token)
